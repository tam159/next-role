# Security Policy

## Supported Versions

We release patches for security vulnerabilities. Which versions are eligible for receiving such patches will be decided on a case-by-case basis.

## Reporting a Vulnerability

If you discover a security vulnerability in NextRole, please report it by:

1. **Emailing the maintainers directly** — Do NOT open a public issue for security vulnerabilities.
2. **Including a detailed description** of the vulnerability, steps to reproduce, and potential impact.
3. **Allowing reasonable time** for a response (typically 48–72 hours) before any public disclosure.

We take all security vulnerabilities seriously and will work to address them promptly.
